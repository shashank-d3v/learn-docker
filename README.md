# Learn Docker

A comprehensive project for learning Docker concepts through practical data engineering examples.

## Project Structure

- **pipeline/**: Data ingestion pipeline for NYC taxi data
- **homework/**: Docker homework assignments and exercises
- **terraform/**: Infrastructure as Code for GCP resources
- **test/**: Test files and utilities

## Getting Started

1. Clone this repository
2. Navigate to the desired directory (pipeline, homework, or terraform)
3. Follow the README instructions in each subdirectory

## Prerequisites

- Docker and Docker Compose
- Python 3.13+
- uv package manager (for Python dependencies)
- Terraform 1.0+ (for infrastructure provisioning)
- Google Cloud Platform account (for cloud resources)

## Topics Covered

- Containerization with Docker
- Multi-service applications with Docker Compose
- Infrastructure as Code with Terraform
- Cloud resource provisioning on GCP
- Data pipelines and ETL processes
- Database management with PostgreSQL
- Web interfaces with pgAdmin
